使用说明：

1. 确保已安装Ollama（https://ollama.com/download）
2. 在命令行运行：ollama serve（确保服务在后台运行，端口11434可用）
3. 下载并运行本程序：NewsClassifier.exe
4. 在输入框中填写新闻标题和内容
5. 点击"开始分类"按钮查看结果

注意：首次运行需要下载qwen3:1.7b模型（约3.5GB），请确保网络畅通



路线图规划 (Roadmap)

本项目是一个持续成长的智能体实验平台。目前的v1.0只是一个起点，未来计划在以下几个方向进行迭代升级：

[v1.1] 视觉与交互升级 (UI/UX)

1. 重构当前较为基础的用户界面，采用更现代化的UI框架（如PyQt现代化主题或基于Web的UI）提升视觉体验。
2. 增加模型加载状态提示和错误日志反馈，避免因环境未配置导致的软件无响应。

[v1.2] 动态分类与提示词工程优化 (Prompt Engineering)

打破固定的“四类新闻”限制：引入动态提示词系统，允许用户在界面上自定义需要分类的新闻标签（例如增加“娱乐”、“游戏”等），智能体将自动重构推理逻辑以适应新的分类体系。

[v2.0] 运行环境解耦 (Deployment & Automation)

解决当前强依赖手动安装Ollama的痛点：开发一键初始化脚本，尝试在软件首次运行时自动检测环境、静默下载Ollama并拉取qwen3:1.7b模型，实现真正的“开箱即用”。
